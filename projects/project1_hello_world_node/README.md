# Project 1: Creation of Docker Containers

### Steps:
1) Create a file that you want to execute with docker
2) Create Dockerfile
3) Build and Run docker file

This project executes a Node.js API that outputs "Hello World" using Docker Containers. 

For this project, I created an API using Node.js, that can be run on the local port 8080.
Then I created a Dockerfile that will 
1) Install required Libraries
2) Copy all the files in the directory
3) Run the file on the port specified
